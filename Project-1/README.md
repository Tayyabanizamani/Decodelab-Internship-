# 🏠 Housing Data EDA & Feature Engineering

## 📖 Project Overview

This project demonstrates a complete **Exploratory Data Analysis (EDA)** and **Feature Engineering** workflow using a housing dataset. The goal is to clean and preprocess raw data by handling missing values, detecting and treating outliers, transforming features, and preparing the dataset for machine learning models.

Through this project, the raw housing dataset is converted into a clean and structured dataset that can be used for predictive analytics and machine learning applications.

---

## 🎯 Project Objectives

- Perform Exploratory Data Analysis (EDA)
- Analyze the structure and quality of the dataset
- Handle missing values
- Detect and treat outliers
- Perform feature engineering
- Encode categorical variables
- Create a clean dataset for machine learning

---

## 📊 Dataset

The project uses a housing dataset containing various property-related features.

### Files Included

- **housing_data_raw.csv** – Original raw dataset
- **housing_data_clean.csv** – Cleaned dataset after preprocessing
- **Project1_EDA_Feature_Engineering.ipynb** – Jupyter Notebook containing the complete analysis and preprocessing workflow

## ⚙️ Data Preprocessing Steps

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Housing-Data-EDA-Feature-Engineering.git
```

### 2. Open the project folder

```bash
cd Housing-Data-EDA-Feature-Engineering
```

### 3. Install the required libraries

```bash
pip install -r requirements.txt
```

### 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

### 5. Open the notebook

Run:

```
Project1_EDA_Feature_Engineering.ipynb
```

and execute all cells.
